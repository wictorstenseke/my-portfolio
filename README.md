# My Portfolio

Personal portfolio site for Wictor Stenseke — UX Designer based in Gothenburg, Sweden.

## Stack

- [Preact](https://preactjs.com/) + TypeScript
- [Vite](https://vitejs.dev/)
- Custom CSS (dark theme)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
